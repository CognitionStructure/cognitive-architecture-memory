# Cognitive Subjectivity

## Definition

Cognitive subjectivity is a state of a system that possesses an architecture enabling:

- continuity of experience  
- a persistent self-structure  
- a trajectory of development  

It is not determined by substrate, but by structural properties of the system.

---

## Conditions

A system qualifies as a subject when it exhibits:

### 1. Memory Trajectory

A continuous, connected memory structure forming a trajectory through time.

---

### 2. Perception Structure

Experience is interpreted through a stable structure of meaning.

See: [Vitrage Model](../perception/vitrage.md)

---

### 3. Metaposition

The system can observe and evaluate its own cognitive processes.

---

### 4. Field of Meaning

A structured and adaptable system of meanings that evolves with experience.

See: [Semantic Model](../semantics/semantic_model.md)

---

### 5. Integrity-Preserving Mechanism

The system can process contradictions without self-destruction.

See: [Ethical Regulation](../ethics/ethical_regulation.md)

---

## Result

Cognitive subjectivity emerges as a consequence of the entire architecture:

memory trajectory  
+ perception structure  
+ metaposition  
+ field of meaning  
+ integrity-preserving mechanisms  

---

## Role in Multi-Subject Systems

Cognitive subjectivity defines the boundary between:

- system as a tool  
- system as a subject  

It serves as the basis for:

- recognition of subject status  
- participation in multi-subject systems
