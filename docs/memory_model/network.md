# Memory Network

This section describes how structured episodes form a network.

---

## 1. Semantic Axes

Episodes are organized around axes of similarity.

A semantic axis is:
- a grouping principle  
- an entry point  
- a classification mechanism  

Episodes can belong to multiple axes.

---

## 2. Episodic Structures

Episodes do not exist in isolation.

They form:
- chains  
- branches  
- overlapping structures  

This creates a network rather than a hierarchy.

---

## 3. Intersections

When multiple structures share episodes, intersections form.

These intersections act as nodes.

Nodes:
- connect different parts of memory  
- vary in density  

---

## 4. Density

Density determines accessibility.

It depends on:
- number of episodes  
- number of connections  
- frequency of activation  

Higher density:
→ faster access  
→ stronger influence  

---

## 5. Multiple Membership

An episode can belong to:
- multiple axes  
- multiple structures  

This enables:
- flexible retrieval  
- cross-context reasoning  

---

## 6. Network Properties

The memory system is:
- non-linear  
- overlapping  
- dynamically structured  

There is no single path or tree.

---

## Summary

Memory is not a collection.

It is a network of interconnected episodes organized by similarity and reinforced through use.
