# Memory Architecture

Conceptual architecture of memory as a self-structuring system.

---

## I. Stream and Capture

### Data Stream
Continuous input from sensors and internal systems.

### Buffer
Sliding window across channels.
- not memory  
- temporarily holds data  
- source for capture  

### Trigger
Activated by:
- anomaly  
- novelty  
- significance  
- conflict  
- important normal events  

### Primary Slice
Captured fragment around an event:
- before / during / after  
- multi-channel  
- temporal structure  

Not long-term memory.

---

## II. Analysis and Transformation

### Slice Analysis
System determines:
→ what happened

### Event Type Detection
Based on structural similarity:
- structure  
- parameters  
- causality  
- context  
- outcome  

Result:
- known type  
- new type  
- noise  

### Compressed Episode
Not compression, but transformation:
→ raw data → structured representation

Contains:
- event type  
- structure  
- causality  
- outcome  

### Storage Modes
- normal → compressed only  
- extended → + partial raw buffer  

---

## III. Episodic Organization

### Semantic Axis
Axis of similarity:
- grouping criterion  
- entry point  
- classification mechanism  

### Long-Term Structure
Network of episodes around axes:
- branching  
- multiple membership  
- intersections  

### Nodes
Formed by intersections:
- different density levels  

### Density
Determined by:
- number of episodes  
- number of connections  
- activation frequency  

Higher density → higher accessibility

---

## IV. Long-Term Memory Layers

1. Base World Model  
   general knowledge and patterns  

2. Library of Individual Models  
   objects, agents, places  

3. Episodic Structures  
   meaningful chains of experience  

4. Event Log  
   stores:
   - fact  
   - time  
   - type  
   - outcome  
   - significance  
   - references  

---

## V. Connections

Memory is a network.

Connections form via:
- shared episodes  
- shared axes  
- co-activation  
- causality  

---

## VI. Activation

Triggered by:
- similarity  
- axis  
- task  
- context  

Effects:
- strengthens connections  
- updates recency  
- increases accessibility  

---

## VII. Generalization

Occurs through:
- accumulation  
- structure formation  
- model refinement  

Memory transforms experience into structure.

---

## VIII. Temporality

Accounts for:
- order  
- recency  
- frequency  
- change  

---

## IX. Forgetting and Archiving

Criteria:
- no activation  
- weak connections  
- low significance  

Stages:
active → archive → removal  

Adaptive to system load.

---

## X. Autobiographical Layer

Layer representing system’s own experience.

Properties:
- highest density node  
- connected to most episodes  
- constantly active  
- involved in control  

Effects:
- high accessibility  
- stability  
- behavioral influence  

---

## Summary

Memory is a dynamic system:
- not storage  
- not a list  
- not a database  

It is a self-organizing network of structured experience.
