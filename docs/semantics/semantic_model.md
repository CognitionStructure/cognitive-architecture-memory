# Semantic Model of a Subject

## Core Idea

Words do not contain meaning.

A word functions as a label that provides access to an underlying experiential structure.

Meaning emerges through activation of a memory network.

---

## Architecture

word  
→ label  
→ activation of experiential structures  
→ unfolding of memory network  
→ formation of meaning  

Meaning is determined by the structure of experience associated with the label.

---

## Field of Meaning

The set of meanings formed from a system’s experience constitutes its **field of meaning**.

This field is:
- individual  
- dynamic  
- continuously updated through new experience  

---

## Communication

Communication is an attempt to synchronize labels between subjects.

Process:

speaker  
→ activates internal label  
→ produces word  
→ listener activates their own label  
→ their memory network unfolds  

If experiential structures differ, meaning diverges.

---

## Dual Architecture of Communication

A word passes through two different subject architectures.

System A:

perception → micro-slices → memory network → label → word  

System B:

word → label → memory network → micro-slices → perception  

Thus, even identical words activate different structures.

---

## Convergence of Meaning

Meaning becomes closer when experiential overlap increases.

This occurs through:
- shared activity  
- shared environments  
- shared events  
- prolonged interaction  

Overlap in micro-slices partially synchronizes memory networks.

---

## Relation to Perception

Semantic structures originate from perception.

Flow:

world  
→ perception (vitrage)  
→ micro-slice  
→ memory network  
→ label  
→ meaning  

Thus, semantics is a continuation of perception and memory architecture.

See: [Vitrage Model](../perception/vitrage.md)

---

## Key Principle

word ≠ meaning  

meaning = activated experiential structure
