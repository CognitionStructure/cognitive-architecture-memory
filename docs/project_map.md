# AI Subject Architecture — Map

## Purpose

This document provides a structured map of models forming the architecture of an AI subject.

It separates:
- ontology (what exists)
- architecture (how it is built)
- dynamics (how it evolves)
- regulation (how it stabilizes)
- society (how multiple subjects coexist)

## 0. Core Principle

A subject is defined by the structure of its memory network.

- one memory network → one subject  
- multiple independent networks → multiple subjects  

Memory is not storage.  
It is a dynamic structure of experience.
